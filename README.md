# free-voices-translator
English:  
A zero-tracking, client-side translation engine built to bypass dogmatic censorship and democratize access to global and LGBTQ+ cinema in Arabic.  

Arabic:
محرك ترجمة لا مركزي لتجاوز مقص الرقابة الدوغمائية وتوفير ترجمات دقيقة للسينما العالمية والكويرية باللغة العربية.
