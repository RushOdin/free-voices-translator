# 🏳️‍🌈 Free Voices Engine (أصوات حرة)



### 🌐 [CLICK HERE TO LAUNCH THE WEB APP / اضغط هنا لفتح التطبيق المباشر] (https://rushodin.github.io/free-voices-translator/)



Information wants to be free. Dogmatic censorship and state-sanctioned moral policing rely on keeping it quarantined. 



**Free Voices** is an open-source, client-side web application engineered to bypass arbitrary media filtering in the MENA region. It democratizes access to marginalized narratives—specifically LGBTQ+ media, feminism, progressive frameworks, and global cinema—by allowing anyone to seamlessly generate high-quality Arabic subtitles without technical expertise.



By moving the translation and file-generation process directly into the user's browser, this tool ensures that human rights and freedom of expression are no longer bottlenecked by archaic laws or localized ISP bans. 



### ⚙️ Core Features

* **Absolute Portability (Single-File Run):** The entire application operates from a single `index.html` file. You can download it and double-click it to run locally on any browser. If authoritarian firewalls block the hosted URL, the file can be shared peer-to-peer and run offline (API features naturally require internet, but processing does not).

* **Zero-Server Architecture:** Runs 100% in your local browser environment. There are no databases, no tracking cookies, and no logs. Your IP and viewing habits remain entirely invisible to state surveillance.

* **Algorithmic RTL Compliance:** Automatically structures Arabic Right-to-Left (RTL) formatting and injects necessary `UTF-8 BOM` encoding so the files render perfectly on VLC, Smart TVs, and standard media players without garbled text.

* **Censorship Bypass:** Provides an empirical, step-by-step workflow for users to obtain unedited source files (via open APIs or AI prompting) and translate them objectively, preserving the original narrative intent.

* **Non-Technical Interface:** Built with psychological safety and ease-of-use in mind. No coding required.



### 🚀 Usage



You have two methods to deploy the engine:



**Method A: Web Access**

Simply click the live application link at the top of this page. No installation required.



**Method B: Local Execution (Anti-Censorship)**

1. Click the green **Code** button on this repository and select **Download ZIP**.

2. Extract the ZIP file to your computer.

3. Double-click the `index.html` file. It will open securely in your default web browser, operating entirely locally.



**The Workflow:**

1. **Acquire:** Follow the built-in guide to obtain raw English `.srt` files from open databases like Subdl or OpenSubtitles.

2. **Translate:** Upload the file. The engine dissects the timestamps and routes the dialogue through translation matrices.

3. **Refine:** Use the side-by-side LTR/RTL interface to manually adjust cultural nuances or correct mistranslations before finalizing.

4. **Compile:** Download the uncompromised Arabic subtitle file and attach it to your media player.

# باللغة العربية (Arabic)



المعلومات ترفض القيود. والرقابة الدوغمائية تعتمد دائماً على حجب المعرفة وعزل الأفراد.



**محرك أصوات حرة** هو تطبيق ويب مفتوح المصدر يعمل كلياً على متصفحك، تم هندسته خصيصاً لتجاوز مقص الرقابة والوصاية الأخلاقية في المنطقة العربية. يهدف المشروع إلى ديمقراطية الوصول للقصص المهمشة—تحديداً المحتوى الكويري (LGBTQ+)، النسوية، والسينما العالمية المتقدمة—من خلال تمكين أي شخص من ترجمة وتوليد ملفات الترجمة بشكل مستقل وبدون أي خبرة تقنية.



### ⚙️ الخصائص التقنية



* **قابلية التشغيل المحلي المطلقة (ملف واحد):** التطبيق مبرمج بالكامل داخل ملف واحد فقط (`index.html`). يمكنك تحميل هذا الملف وتشغيله محلياً بالنقر عليه مرتين ليفتح في متصفحك. إذا قامت الحكومات بحجب الرابط الإلكتروني، يمكن مشاركة هذا الملف مباشرة عبر التطبيقات الآمنة وتشغيله دون الحاجة لاستضافات الويب.

* **معمارية خالية من الخوادم (Zero-Server):** التطبيق يعمل 100% على جهازك المحلي. لا يوجد تتبع، لا تسجيل للبيانات، ولا خوادم خلفية. هويتك وما تشاهده يبقى مخفياً تماماً عن أي رقابة حكومية أو مزودي الخدمة.

* **دعم تلقائي للغة العربية (RTL):** الخوارزمية تعالج اتجاه النص من اليمين لليسار، وتقوم بحقن ترميز `UTF-8 BOM` برمجياً لضمان ظهور الترجمة العربية بشكل سليم على شاشات التلفاز الذكية وبرامج مثل VLC دون أن تتحول لرموز غير مفهومة.

* **واجهة بديهية للمستخدم العادي:** مصمم ليكون مرشداً متكاملاً. يأخذ بيدك من مرحلة البحث عن الملفات الأصلية، مروراً بالترجمة الآلية، وصولاً إلى التدقيق والمشاهدة.
